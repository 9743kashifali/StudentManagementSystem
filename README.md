# StudentManagementSystem

A Java-based Student Management System with OOP principles, Swing/AWT GUI, custom exception handling, and support for Undergraduate & Graduate student records.

## Features
- Add Undergraduate & Graduate students
- GPA validation with custom exception
- Swing GUI with AWT controls
- Search student by ID

## Concepts Covered
- Encapsulation, Inheritance, Polymorphism
- Packages & Exception Handling
- Swing & AWT GUI
- Event Handling

## How to Run
1. Open in NetBeans
2. Run `project.app.Main`

## Tech Stack
Java | Swing | AWT | NetBeans
